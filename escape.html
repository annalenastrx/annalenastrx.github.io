OVERVIEW

The VR Coding Escape Room was created to arouse interest in programming as well as create an immersive and fun experience. With the help of VR headsets, users enter a digital escape room where they have to find code snippets in the room and put them together to eventually escape. On code completion, users get deeper explanation of concepts to strengthen the learning effect. Correctly completed code puzzles also have an influence on the virtual world to explain coding concepts less abstract. For example in one level, a correctly composed code turns on the light. The difficulty of the code puzzles increases with the number of completed puzzles.

MY ROLE

Group project with two fellow students. Together we developed the concept for the escape room. I was responsible for the Steam VR and XR Toolkit integration as well as implementing the UI and interactions, including picking up code snippets and ordering them.
